## DevCamper API

>Bootcamp API for DevCamper Application, which is a Bootcamp directory website

## Usage
Rename "config/config.env.env" to "config/config.env" and update the value/settings on your own

## Install Depencies
```
npm install
```

##Run app
```
# Run in dev Mode
npm run dev

# Run in prod Mode
npm start
```
